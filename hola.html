Manual de GIT

git init; Activas git dentro de cualquier directorio

git add -A(TODOS): Asigna el archivo que deseas subir dentro del carpetita

git commit -m "": Nos permite asignar un mensaje para el archivo

git nombre de la URL para subir mis archivos

git push origin master: Subes tus archivitos

git log nos permite ver los commits dentro de la inicializacion
